About This Project

This microproject was created for COMP2140 to practice using Node.js and Express to build a small website with a JSON API. It includes:

  • An Express server running on localhost:3000
  • A static website inside the public folder
  • A JSON file stored in data/courses.json
  • An API endpoint that returns course data in JSON format
  • A front-end page that fetches and displays the JSON data
